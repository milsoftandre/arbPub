0. Если сервер голый нужно установить Apache, MySQL, PHP для работы панели
https://www.digitalocean.com/community/tutorials/how-to-install-linux-apache-mysql-php-lamp-stack-on-ubuntu-22-04

1.  Распаковываем архив в папку веб сервера. Создаем базу MySQL

2. Открываем файл конфигурации  .env
Отредактировать достаточно:

APP_URL=http://181.41.141.146/ - путь 
DB_DATABASE=1arb – имя базы
DB_USERNAME=andre – имя пользователя MySQL
DB_PASSWORD=5jXK_XKXKnyc – пароль пользователя MySQL


3.  Так же изменить путь в config/app.php

4.  Вложенным папкам storage – дать права на запись. 

5. В терминале зайти в папку проекта и выполнить команду
php artisan migrate

Это выполнит миграцию базы данных и создаст базу. Если все сделано верно – должен работать вход в кабинет с базовым паролем админа.

6. В терминале выполнить команду
crontab -e

В конце файла добавить:

*/20 * * * * curl http://181.41.141.146/updatestart

Эта команда для планировщика который будет каждые 20 минут проверять работу ботов и возобновлять если они вдруг остановились. 

7. В терминале выполнить команду
pip3 install python-binance

Это устанавливет библиотеку бинанса для работы бота.
